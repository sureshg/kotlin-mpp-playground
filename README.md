### Install OpenJDK 21

```bash
# Mac OS
$ curl -s "https://get.sdkman.io" | bash
$ sdk i java 21.ea.23-open
$ sdk u java 21.ea.23-open
```

### Build & Run

```bash
$ ./gradlew build
```

For doing the release build, go to [build job][build-job] and click `Release Build`
